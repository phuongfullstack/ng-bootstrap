Core Dropdown Component Demo

Component dropdown tùy chỉnh với Bootstrap 5.3, hỗ trợ Reactive Forms, validation và nhiều tính năng khác.

1. Basic Dropdown (Required)
Tính năng:
  • Required validation
  • Dynamic placeholder
  • Value change event
  • Focus/Blur events
2. Dropdown with Hint
Tính năng:
  • Optional field
  • Hint text hỗ trợ
  • Selection change event
3. Multiple Selection
Tính năng:
  • Multiple selection
  • Array value binding
  • Ctrl+Click để chọn nhiều
4. Disabled Options
Tính năng:
  • Disabled options trong list
  • Pre-selected value
  • Option "Không hoạt động" bị disabled
5. Dynamic Size
Tính năng:
  • Size tự động thay đổi theo giá trị chọn
  • xs, s → sm size
  • m → normal size
  • l, xl → lg size
6. Disabled Dropdown
Tính năng:
  • Disabled state
  • Form control disabled
